# Expense Tracker

This is a Personal Expense Tracker web application developed as part of the COMP 2154 – System Development Project course.

## Project Description
The application helps users track their income, expenses, and credit card balances. It allows users to record transactions and view summaries of their financial activity.

## Technologies Used
- C#
- ASP.NET Core MVC
- PostgreSQL
- Git & GitHub
